## Projects

Projects helps you to organize your collections and NFT drops. 

Free users, only have acces to one project.

The project object have the following structure:

```javascript
 Project: {
            companyId:  
            createdAt:  
            createdBy: 
            description:
            name:
            network:
            updatedAt: 
            __v:
            _id:
            
        }
```
